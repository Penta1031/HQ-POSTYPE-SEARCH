검색기 이용 가이드

1. 다중 검색 지원
검색창에 작품명, 작가명, 장르, 키워드 등을 자유롭게 입력해 보세요.

2. 상세 필터 기능
[상세 필터] 탭에서 원하는 결제/연령/장르/키워드를 복합적으로 적용할 수 있습니다. (장르/키워드는 중복 선택 가능)

3. 스포일러 (엔딩) 확인
엔딩 정보는 가려져 있습니다. 작품 목록의 엔딩 확인 버튼을 누르면 결말을 볼 수 있습니다.

4. 관리자 모드
우측 상단의 자물쇠() 아이콘을 누르면 작품 데이터를 직접 추가/수정/삭제하거나 필터 옵션을 커스텀할 수 있습니다.
